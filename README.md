# poedat

Go version of [poe-dat-viewer](https://github.com/SnosMe/poe-dat-viewer) cli.

Export table as json from dat file.

# Usage:

```
Usage of ./poedat.exe:
  -d string
        the dat file path
  -g int
        the game version, poe1 is 1, poe2 is 2 (default 1)
  -s string
        the schema file path
  -t string
        the table name
```
